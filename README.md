# Student Budget App

Приложение для учёта личных расходов студента. Помогает контролировать бюджет, отслеживать траты по категориям и не выходить за лимиты.

## Структура репозитория

## Идея проекта
Студенты часто сталкиваются с нехваткой денег в конце месяца.  
**Student Budget App** — простое и удобное решение для ведения личного бюджета без лишней сложности.

**Ключевые возможности:**
- Быстрая запись расходов (3-5 секунд)
- Наглядная статистика по категориям
- Установка месячных лимитов
- Доступ с телефона и компьютера

## Технический стек (план)
- **Frontend (Web)**: HTML, CSS, JavaScript (React)
- **Mobile**: Kotlin / Swift или Flutter
- **Backend**: Python (FastAPI / Django) или Node.js
- **DB**: PostgreSQL
