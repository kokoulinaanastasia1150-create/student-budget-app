# Серверный API

## Назначение папки
В этой папке находится бэкенд-часть проекта: серверное приложение, база данных и бизнес-логика.

## Планируемые компоненты
- **REST API** — эндпоинты для авторизации, операций, категорий, статистики
- **База данных** — PostgreSQL / SQLite для хранения пользователей и транзакций
- **Аутентификация** — JWT-токены, хеширование паролей
- **Бизнес-логика** — расчёт остатков, лимитов, аналитика
- **Логирование** — запись действий и ошибок

## Взаимодействие с другими частями
- Принимает запросы от **MobileApp** и **Web**
- Возвращает данные в формате JSON
- Единая точка доступа для всех клиентских приложений
