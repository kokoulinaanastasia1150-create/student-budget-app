# Веб-приложение

## Назначение папки
В этой папке находится исходный код веб-версии приложения для учёта расходов.

## Планируемые компоненты
- **Страница входа** — авторизация пользователей
- **Дашборд** — сводка по бюджету на текущий месяц
- **Таблица операций** — просмотр, фильтрация, редактирование расходов
- **Категории** — управление статьями расходов
- **Отчёты** — визуализация трат за произвольный период

## Взаимодействие с другими частями
- Отправляет запросы к **ServerApi** через AJAX/Fetch
- Отображает те же данные, что и **MobileApp**, но в адаптивном веб-интерфейсе
- Не имеет прямой связи с мобильным приложением — только через сервер
