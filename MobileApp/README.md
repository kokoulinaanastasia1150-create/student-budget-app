# Мобильное приложение

## Назначение папки
В этой папке находится исходный код мобильного приложения для студентов.

## Планируемые компоненты
- **Экран авторизации** — вход/регистрация через email или соцсети
- **Главный экран** — отображение текущего баланса и последних операций
- **Добавление расходов** — быстрая запись трат по категориям
- **Статистика** — графики и диаграммы расходов за период
- **Настройки** — управление профилем и бюджетными лимитами

## Взаимодействие с другими частями
- Отправляет HTTP-запросы к **ServerApi** (REST API)
- Получает данные о пользователе, транзакциях и категориях
- Отправляет уведомления при превышении лимита бюджета
- **Web** и **MobileApp** используют общий API и имеют похожий функционал
